## 使用案例

| 使用案例 | 個人檔案作業 |
| ----- | ----- |
| 使用案例描述 | 人員可以預覽會員的個人檔案，會員可以觀看其他會員的公開個人檔案、自己的個人檔案可選擇不公開，預設公開、可以修改個人檔案、可以看到自己的貼文、粉絲、追蹤者，系統管理員可以觀看所有人的個人檔案 |
| 主要參與者 | 人員、會員、系統管理員 |
| 利害關係人與目標 | 人員: 能預覽會員的個人檔案 |
|   | 會員: 可觀看其他會員的公開個人檔案、可選擇不公開檔案、可修改個人檔案、可看到自己的貼文、粉絲、追蹤者 |
|   | 系統管理員: 可觀看所有人的個人檔案 |
| 前置條件 | 會員、系統管理員須先登入 |
| 後置條件 | 選擇不公開檔案後，系統會將公開轉成私人 |
| 主要成功情節 | 1.人員可點擊會員帳號進入個人檔案 |
|   | 2.會員登入後，點擊個人檔案 |
|   | 3.人員可預覽該會員的個人檔案 |
|   | 4.點擊個人檔案後可以看到自己的貼文、粉絲、追蹤者 |
|   | 5.點擊編輯個人檔案可修改個人檔案 |
|   | 6.若點擊隱私設定和帳號安全可將帳號設為不公開 |
|   | 7.若點擊其他會員帳號後可觀看該會員的公開個人檔案 |
|   | 8.系統管理員登入後可看到所有會員帳號 |
|   | 9.點擊會員帳號後可以看到該會員的個人資料 |
| 例外情節 | 6a.若不選擇不公開帳號，則個人檔案繼續公開 |
|   | 7a.若該會員有設定不公開個人檔案，則系統對未追蹤的會員或人員顯示此個人檔案不公開 |
| 其他需求 | 點擊粉絲後除了可以看到已追蹤的用戶之外也可以看到系統推薦的用戶 |

| 使用案例 | 搜尋作業 |
| ----- | ----- |
| 使用案例描述 | 會員、系統管理員可以搜尋其他會員，可以查看自己的搜尋紀錄 |
| 主要參與者 | 會員、系統管理員 |
| 利害關係人與目標 | 會員: 搜尋其他會員、看到自己的搜尋紀錄 |
|   | 系統管理員: 搜尋其他會員、查看該會員的搜尋紀錄 |
| 前置條件 | 須完成登入 |
| 後置條件 | 搜尋後可以在搜尋欄位顯示搜尋紀錄 |
| 主要成功情節 | 1.登入後進入首頁 |
|   | 2.點擊搜尋欄位 |
|   | 3.輸入會員id搜尋想找的會員 |
|   | 4.點擊該會員可以查看該會員 |
|   | 5.返回首頁再次點擊搜尋會出現之前的搜尋紀錄 |
| 例外情節 | 3a.若搜尋結果沒有出現會員，系統顯示無搜尋結果 |
|   | 3b.若從未搜尋過，系統顯示沒有近期搜尋紀錄 |
| 其他需求 | 不須完整寫入全部會員id，會自動顯示相似的會員供參考 |

|   | 當為專業帳號姓名、用戶名稱、電話、電子郵件其中一個為空時，系統會顯示此為必填欄位 |
